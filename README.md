# Substack Archiver

A Chrome/Brave extension to save Substack articles as self-contained HTML files for offline reading.

## Features

- **Download HTML** from the popup — article is saved as a single file with all styles inlined
- **Editable filename** — pre-filled from the article title, change it before saving
- **Right-click → Export page as HTML** — works on any page, opens native Save As dialog
- Paywalls, modals, and subscribe widgets are stripped from the archive
- Embedded YouTube/Vimeo videos stay responsive

## Install (unpacked)

1. Clone or download this repo
2. Go to `chrome://extensions` → enable **Developer mode**
3. Click **Load unpacked** → select the repo folder

## Usage

- Navigate to a Substack article
- Click the extension icon, optionally rename the file, then hit **Download HTML**
- Or right-click anywhere on the page and choose **Export page as HTML**
